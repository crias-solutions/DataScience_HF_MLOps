# Image Dataset

This directory should contain subdirectories for each class.

## Structure

```
image/
├── positive/
│   ├── image1.jpg
│   ├── image2.png
│   └── ...
└── negative/
    ├── image1.jpg
    ├── image2.png
    └── ...
```

## Usage

Place your images in subdirectories named after each class. The folder names will be used as labels for classification.

Supported image formats: .jpg, .jpeg, .png, .bmp, .gif
